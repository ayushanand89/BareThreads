Under the hood

Secure & serverless.

Serverless-safe data layer
A cached MongoDB connection is awaited inside each request, so the API connects reliably on Vercel's serverless runtime — fixing cold-start failures.
Auth, two ways
JWT in cross-site httpOnly cookies, plus Google sign-in verified server-side from a Google ID token — both issue the same session.
Authorization at the controller
Ownership checks scope every order and checkout to its owner; admin routes are gated. IDOR holes closed, writes field-whitelisted.
Correct commerce
Guest→user cart merge, server-computed ratings, discount pricing and stock validation — with a consistent JSON error contract.
